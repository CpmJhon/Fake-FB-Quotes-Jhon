Cara Penggunaan QuickFake

Ikuti langkah mudah berikut untuk membuat mockup impianmu

01

Pilih Format

Pilih format mockup yang diinginkan dari 13+ pilihan seperti Brat Text, Fake Call, WA Group, IG Post, TikTok Profile, TikTok Comment, YT Community, YT Comment, dan lainnya.

02

Isi Parameter

Masukkan data sesuai format yang dipilih. Setiap format memiliki parameter unik seperti teks, nama, URL gambar, jumlah like, dll.

03

Generate

Klik tombol "Generate Sekarang" dan tunggu proses pembuatan mockup (hanya beberapa detik).

04

Download

Hasil mockup akan tampil. Klik tombol "Download Gambar" untuk menyimpan ke perangkatmu secara otomatis.

Tips & Trik

  • SSWeb: Gunakan skala 2x atau 3x untuk hasil HD
  • SSWeb: Pilih "Halaman Penuh: Ya" untuk screenshot seluruh halaman website
  • YT Comment: Pastikan URL avatar valid dan dapat diakses publik
  • Fake Call: Durasi bisa diisi format seperti "19:31" atau "2:45"
  • Untuk format media sosial, isi angka like/comment dengan format numerik biasa
  • Semua hasil generate bersifat real-time dan langsung bisa didownload
  • Jika download tidak berjalan, klik kanan pada gambar lalu pilih "Save image as..."
  • Gunakan URL gambar dari hosting gratis seperti imgur.com atau postimages.org untuk hasil terbaik

Panduan Lengkap QuickFake

Penjelasan detail setiap fitur dan penggunaannya

SSWeb (Screenshot Website)

Fitur untuk mengambil screenshot dari website apapun secara instan.

Parameter: URL (wajib), Perangkat (Desktop/Mobile/Tablet), Halaman Penuh (Ya/Tidak), Skala (1x/2x/3x)
Tips: Gunakan skala 2x atau 3x untuk hasil HD, pilih "Halaman Penuh: Ya" untuk screenshot seluruh halaman website

Brat Text

Membuat teks dengan gaya aesthetic ala Brat. Cukup masukkan teks apapun, dan sistem akan menghasilkan gambar dengan typography keren dan efek gradien modern.

Parameter: Teks
Contoh: Text: "Halo semesta!" → Hasil teks dengan gaya bold dan warna gradien

Fake Call

Simulasi tampilan panggilan masuk dengan avatar custom. Cocok untuk konten storytelling, meme, atau konten lucu.

Parameter: Nama penelepon, Durasi panggilan, URL Avatar

Fake Channel

Membuat tampilan channel YouTube dengan thumbnail, nama channel, jumlah subscriber, deskripsi, dan tanggal.

Parameter: URL Gambar, Nama Channel, Followers, Deskripsi, Tanggal

Fake DANA

Simulasi tampilan saldo DANA untuk keperluan kreatif, promosi, atau konten edukasi.

Parameter: Nominal (contoh: Rp 100.000)

FB Comment

Komentar Facebook dengan nama, komentar, dan foto profil custom.

Parameter: Nama, Komentar, URL Foto Profil

WA Group & WA Group V2

Membuat screenshot grup WhatsApp dengan berbagai variasi. V2 memiliki tampilan lebih modern dengan info member detail dan author.

Parameter V1: URL Icon, Judul, Jumlah Pesan, Waktu
Parameter V2: URL Avatar, Nama Admin, Jumlah Member, Deskripsi, Author, Tanggal

IG Post

Mockup postingan Instagram dengan like, comment, share, repost, dan caption. Bisa custom avatar dan konten.

Parameter: Avatar URL, Konten URL, Username, Likes, Comments, Share, Repost, Date, Caption

TikTok Profile & TikTok Comment

Simulasi profil TikTok dengan informasi following, followers, likes, dan komentar dengan tampilan autentik.

Profile: Nama, Username, Following, Followers, Likes, URL Foto
Comment: URL Foto, Username, Komentar, Tanggal

YouTube Community

Postingan komunitas YouTube dengan like dan comment count. Cocok untuk simulasi konten viral.

Parameter: Avatar URL, Konten URL, Username, Deskripsi, Like, Comment

YT Comment (YouTube Comment)

Membuat komentar YouTube dengan avatar custom. Cocok untuk simulasi komentar di video YouTube.

Parameter: Username, Komentar, URL Avatar
Contoh: Username: "cpmjhon21", Komentar: "Kontennya keren banget!", Avatar: "https://example.com/avatar.jpg" → Hasil komentar YouTube dengan foto profil dan teks komentar

Informasi Penting

  • Semua gambar yang dihasilkan bersifat sementara dan tidak disimpan di server
  • Gunakan URL gambar yang valid (pastikan gambar dapat diakses publik)
  • Website ini gratis dan hanya untuk keperluan kreatif & edukasi
  • Hasil generate dapat digunakan untuk konten media sosial, presentasi, atau desain
  • Fitur YT Comment menggunakan API dari deline.web.id
  • Fitur SSWeb dan Mockup lainnya menggunakan API dari zenzxz.my.id

Arti QuickFake

QuickFake adalah istilah yang biasanya digunakan untuk menggambarkan konten palsu yang dibuat dengan cepat menggunakan teknologi digital, terutama berbasis AI.

Secara sederhana:
• 👉 QuickFake = Fake (palsu) + Quick (cepat dibuat)

Penjelasan lengkap QuickFake merujuk pada:
• Gambar, video, atau audio palsu
• Dibuat dalam waktu singkat
• Menggunakan teknologi seperti AI
• Sering dipakai untuk hiburan, prank, atau bahkan hoaks

Contoh:
• Foto editan wajah seseorang dalam situasi yang tidak pernah terjadi
• Video singkat manipulasi ekspresi wajah
• Suara tiruan seseorang yang dibuat cepat

Kesimpulan:
QuickFake adalah konten manipulasi digital yang dibuat secara instan atau cepat, biasanya menggunakan bantuan teknologi AI atau aplikasi editing modern.

Kontribusi & Pengembangan

QuickFake bersifat open-source. Kamu bisa mengembangkan dengan:

  • Menambahkan format mockup baru dengan menambahkan object di array formats di script.js
  • Mengustomisasi tampilan di file style.css
  • Menambahkan fitur export ke format lain (PDF, PNG, dll)
  • Menambahkan fitur share langsung ke media sosial

⚠️ Peringatan Penting: Gunakan website ini secara bijaksana. Tidak untuk menipu atau merugikan orang lain. Developer - JHON PRODUCTION tidak bertanggung jawab jika website ini disalahgunakan. Website ini hanya untuk keperluan edukasi dan kreatif. Terimakasih.

Format Data (formats array)

{
    id: 'brat',
    name: 'Brat Text',
    icon: 'fas fa-font',
    endpoint: '/maker/brat',
    baseUrl: 'https://api.zenzxz.my.id',
    params: [{ name: 'text', label: 'Teks', type: 'text', placeholder: '...' }]
}

// Format YT Comment (dari deline.web.id)
{
    id: 'ytcomment',
    name: 'YT Comment',
    icon: 'fab fa-youtube',
    endpoint: '/maker/ytcomment',
    baseUrl: 'https://api.deline.web.id',
    params: [
        { name: 'username', label: 'Username', type: 'text', placeholder: 'cpmjhon21' },
        { name: 'text', label: 'Komentar', type: 'textarea', placeholder: 'Tulis komentar...' },
        { name: 'avatar', label: 'URL Avatar', type: 'url', placeholder: 'https://example.com/avatar.jpg' }
    ]
}

Setiap format memiliki struktur yang sama dengan properti: id, name, icon, endpoint, baseUrl (opsional), dan params (array parameter).
